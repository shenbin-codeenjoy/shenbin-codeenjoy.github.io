Markdown 语法新姿势

发表于2021-02-12|更新于2021-02-12|GrammerMarkdown

      Markdown 是一种易读易写的纯文本「 标记语言 」。它通过简洁的语法将纯文本转换成 XHTML（ 或者 HTML ）文档，获得了很多写作爱好者、程序员的喜爱。

一、Markdown 特点

1.1 优点

  • 具有一定的跨平台性
          在支持 Markdown 语法的平台中，同样的文字可以实现同样的效果。因此，基于 Markdown 编写文章的写作爱好者、程序员可以专心的撰写优质文章，而不用为文章排版等问题而困扰。

  • 支持图片插入、超链接、锚点
          由于 Markdown 语法的最终呈现方式是 XHTML（ 或者 HTML ）文档，因此它支持图片、超链接、锚点等一系列功能。

1.2 缺点

  • 不是所有平台都支持 Markdown 语法
          Markdown 语法虽然目前获得了很多写作平台的支持，但是也有少部分平台暂时不支持 Markdown 语法。
